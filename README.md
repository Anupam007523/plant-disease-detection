# 🌿 AI Integrated Plant Disease Detection System

An intelligent web-based system that detects plant diseases using AI and provides prevention methods and fertilizer suggestions.

---

## 🚀 Features

- 📸 Upload plant leaf images
- 🤖 AI-based disease detection
- 🧠 Accurate prediction using trained model
- 💡 Prevention tips for diseases
- 🌱 Fertilizer suggestions
- 📱 Simple and user-friendly UI

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, Bootstrap  
- **Backend:** Python (Flask)  
- **Machine Learning:** CNN Model  
- **Other Tools:** OpenCV, NumPy
